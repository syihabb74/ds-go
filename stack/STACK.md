# STACK

## STACK COMPONENT

- ITEMS -> it could be anything for example string, int, another struct

## METHOD 

- Append -> Adding item in the stack it will be adding in the end position (len(items))
- Pop -> Remove item in the stack it will be remove item from the end of items 

## BEHAVIOR
- LIFO (Last In First Out)

## BENEFITS
- Clear control sequential of data
- Operation Push And Pop Very Fast O(n)

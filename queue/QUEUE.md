# QUEUE 

## QUEUE COMPONENT

- ITEMS -> it could be anything for example string, int, another struct

## METHOD 

- Enqueue -> Adding item in the queue it will be adding in the end position (len(items))
- Dequeue -> Remove item in the queue it will be remove item from the end of items 

## BEHAVIOR
- FIFO (First In First Out )

## BENEFITS
- Fair Processing (First data in it will be guarantee priority first)
- Avoiding starvation (Old data / element never processing)
